# Marketing Media Assets Requirements

To execute a flawless public launch, the following high-resolution assets must be generated and stored in the `/public/assets/marketing` directory before deployment.

## 1. Website Hero (Cinematic)
- **Filename**: `marketing-hero.webp`
- **Dimensions**: `2560 x 1440` (16:9)
- **Usage**: Primary PR image, press releases, high-res displays.
- **Content**: The space-themed horizon background featuring the "Engineering the Future" typography and floating blue particles.

## 2. Homepage Screenshot
- **Filename**: `showcase-homepage.webp`
- **Dimensions**: `1920 x 1080` (16:9)
- **Usage**: Dribbble, Behance, Portfolio showcases.
- **Content**: A full desktop viewport capture of the initial scroll state, highlighting the glassmorphic navbar and primary CTAs.

## 3. Services Screenshot
- **Filename**: `showcase-services.webp`
- **Dimensions**: `1920 x 1080` (16:9)
- **Usage**: LinkedIn carousels, B2B marketing.
- **Content**: The Services grid showing the hover states of the AI, Web, and Mobile development cards.

## 4. CryptiqVerse Showcase
- **Filename**: `showcase-cryptiqverse.webp`
- **Dimensions**: `1920 x 1080` (16:9)
- **Usage**: X (Twitter) launch thread, Case Study promotion.
- **Content**: The CryptiqVerse project page hero section with the live URL button visible.

## 5. Design System
- **Filename**: `showcase-design-system.webp`
- **Dimensions**: `1920 x 1080` (16:9)
- **Usage**: Developer communities (Dev.to, Hashnode, GitHub).
- **Content**: The `/design-system` route displaying the color tokens, button variants, and typography scale.

## 6. Mobile Preview
- **Filename**: `showcase-mobile.webp`
- **Dimensions**: `1080 x 1920` (9:16)
- **Usage**: Instagram Stories, X (Twitter) mobile highlights.
- **Content**: An iPhone 15 Pro mockup displaying the responsive mobile menu and perfectly scaled typography.

## 7. Desktop Preview
- **Filename**: `showcase-desktop-mockup.webp`
- **Dimensions**: `2560 x 1440` (16:9)
- **Usage**: Website hero sections, landing page conversions.
- **Content**: A MacBook Pro mockup displaying the Exotic Co. portfolio floating in a dark environment.

## 8. Social Preview (OpenGraph / Twitter Card)
- **Filename**: `og-image.webp` (must be converted to `og-image.jpg` or `png` for meta tags)
- **Dimensions**: `1200 x 630` (1.91:1)
- **Usage**: Slack previews, iMessage previews, Twitter/LinkedIn link unfurling.
- **Content**: The Exotic Co. Logo centered on the deep space background with the tagline "Engineering the Future" clearly legible in the center.

> **Technical Note:** All images must be compressed using WebP format for the website, but fallback JPGs/PNGs must be generated for OpenGraph and external PR distribution where WebP is not fully supported.
