# Client Onboarding Checklist

This checklist ensures a seamless transition from the Sales phase to the Engineering phase for all new Exotic Co. enterprise clients.

## Phase 1: Contract & Admin
- [ ] Master Services Agreement (MSA) signed.
- [ ] Statement of Work (SOW) signed.
- [ ] Initial deposit / retainer invoiced and cleared.
- [ ] Non-Disclosure Agreement (NDA) executed (if applicable).
- [ ] Client added to Exotic Co. CRM.

## Phase 2: Access & Environments
- [ ] Receive access to client's existing GitHub/GitLab repositories.
- [ ] Receive access to client's AWS/GCP/Vercel environments.
- [ ] Receive API Keys, environment variables, and third-party credentials.
- [ ] Create shared Slack/Teams channel for engineering communication.
- [ ] Setup Jira / Linear project board and invite client stakeholders.

## Phase 3: Brand & Assets
- [ ] Collect high-resolution vector logos (SVG/AI).
- [ ] Collect Brand Guidelines (Colors, Typography).
- [ ] Audit existing Figma files / UI designs.
- [ ] Request necessary copywriting / marketing assets.

## Phase 4: Discovery Kickoff
- [ ] Schedule 90-minute Technical Discovery Call with CTO/Founders.
- [ ] Complete the *Discovery Call Questionnaire*.
- [ ] Finalize the Technical Architecture Document (TAD).
- [ ] Establish weekly sprint review meeting schedule.
- [ ] Formally kick off Sprint 0.
