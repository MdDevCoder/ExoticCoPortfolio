# Changelog

All notable changes to the **Exotic Design System** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-01

### Added
- Centralized Design Tokens (`colors.ts`, `spacing.ts`, `motion.ts`).
- Standardized typography components (`Display`, `H1-H6`, `Body`).
- Core primitive UI components (`Button`, `Card`, `Badge`, `Input`, etc.).
- Layout components (`Container`, `Section`, `Grid`).
- `/design-system` interactive documentation and playground.
- Framer Motion physics-based easing presets (`exoticMotion`).

### Changed
- Shifted all portfolio component styling away from hardcoded utility classes to formal design system tokens.
- Standardized all application border-radiuses to the new token scale (`2rem` and `2.5rem`).
